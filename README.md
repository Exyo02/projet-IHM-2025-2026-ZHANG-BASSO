## Description de l’application

**CampuStage** est un prototype d’interface permettant à des étudiants de :
- consulter des offres de stage ;
- gérer une liste de favoris ;
- naviguer dans une interface claire et ergonomique ;
- accéder aux fonctionnalités après authentification.

---

## Technologies utilisées
- **Blazor WebAssembly**
- **HTML / CSS**
- **Git & GitHub**
- **Canva / Draw.io** (conception et modélisation)
